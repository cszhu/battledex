# Battledex
Chrome extension that looks up a Pokemon. Basically a convenient Pokedex. 

This Pokedex is able to look up:
  - Name & ID #
  - Type(s)
  - Ability
  - Height
  - Weight
  - PokeDex description
  - Sprite
  
If you mistype the Pokemon name it will display an error.
